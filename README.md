# ESP32-Based Industrial Automation System

This repository contains the simulation model for a solar-powered ESP32-based industrial automation system with:

- Real-time monitoring (Voltage, Current, Temperature, Humidity)
- Fault detection (Overcurrent, Undervoltage, Overheat, Humidity)
- Priority-based load control
- Solar tracking using LDR sensors
- OLED display and serial monitoring

## Simulation Features

- Normal operation scenario
- Overcurrent fault
- Undervoltage trip
- Overheat shutdown
- High humidity trip
- Overload demand (priority-based control)
